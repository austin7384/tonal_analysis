INPUTS
------

OPERATIONS
----------
- Defines decade categories (1950-59, 1960-69, ..., 2010-15)
- Collapses data to article count by Decade x Journal
- Reshapes to wide format with journals as columns and decades as rows
- Adds a totals row and a totals column
- Converts to a Stata matrix for LaTeX export
